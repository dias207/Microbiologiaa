# 🦠 Эксклюзивная Система Определения Бактерий

## 🎯 Описание

Эксклюзивная система классификации бактерий, работающая только с 19 заранее определенными изображениями. Система использует компьютерное зрение для точного определения таксономии бактерий и защищена от распознавания посторонних изображений.

## 🏥 Особенности

- 🔒 **Эксклюзивная база**: Только 19 определенных бактерий
- 🎯 **Точность**: 20-98% на обученных данных
- ❌ **Защита**: Посторонние изображения определяются с ошибкой
- 🦠 **Авто-анализ**: Определение количества палочек
- 🧬 **Таксономия**: Полная классификация (семейство, род, вид)

### Основные возможности:
- **Классификация бактерий** по трем уровням таксономии:
  - Тұқымдастық (Family)
  - Туыстастық (Genus) 
  - Түрі (Species)

- **Детекция палочковидных бактерий**:
  - Автоматическое обнаружение палочек (bacilli)
  - Подсчет количества обнаруженных палочек
  - Визуализация обнаруженных объектов

- **Визуализация внимания модели**:
  - Grad-CAM для демонстрации областей внимания
  - Помогает понять, на что обращает внимание модель

### Пример вывода:
```
Тұқымдастық: Micrococaceae (95.2%)
Туыстастық: Staphylococcus (87.6%)
Түрі: S. epidermidis (92.1%)
Обнаружено палочек: 17
```

## � Список бактерий в базе

1. **S. aureus** - гроздевидные кокки
2. **N. meningitidis** - диплококки
3. **V. cholerae** - изогнутые запятые
4. **S. pyogenes** - цепочки кокков
5. **B. melitensis** - очень мелкие кокки
6. **E. coli** - кишечные палочки
7. **B. anthracis** - крупные палочки со спорами
8. **C. tetani** - палочки с терминальными спорами
9. **C. jejuni** - S-образные изогнутые палочки
10. **S. dysenteriae** - шигеллез (грамотрицательные палочки)
11. **F. tularensis** - очень мелкие коккобактерии
12. **Y. pestis** - чумные палочки с биполярным окрашиванием
13. **L. interrogans** - тонкие спиралевидные бактерии
14. **M. leprae** - проказа (кислотоустойчивые палочки)
15. **C. botulinum** - ботулизм (очень крупные палочки с большими спорами)
16. **H. pylori** - хеликобактериоз (спиралевидные с жгутиками)
17. **S. typhi** - брюшной тиф (палочки с перитрихиальными жгутиками)
18. **N. gonorrhoeae** - гонорея (диплококки - пары кофейных зерен)
19. **C. perfringens** - газовая гангрена (очень крупные палочки с терминальными спорами)

## �🛠️ Технический стек

- **PyTorch** - фреймворк машинного обучения
- **ResNet18** - архитектура CNN (transfer learning)
- **OpenCV** - компьютерное зрение для детекции палочек
- **Streamlit** - веб-интерфейс
- **Grad-CAM** - визуализация внимания модели
- **Albumentations** - аугментация изображений

## 📊 Использование

### Через веб-интерфейс:
1. Откройте приложение в браузере
2. Загрузите микроскопическое изображение
3. Настройте параметры детекции (опционально)
4. Получите результаты классификации и детекции

### Программно:
```python
from model import create_model, load_model_checkpoint
from utils import BacilliDetector, ImageProcessor

# Загрузка модели
model = create_model()
model = load_model_checkpoint(model, 'best_model.pth')

# Обработка изображения
detector = BacilliDetector()
processor = ImageProcessor()

image = cv2.imread('bacteria_image.jpg')
bacilli, count = detector.detect_bacilli(image)
input_tensor = processor.preprocess_for_model(image)

# Получение предсказаний
with torch.no_grad():
    predictions = model(input_tensor)
```

## 🔧 Настройка параметров детекции

В веб-интерфейсе можно настроить:
- **Минимальное соотношение сторон** для палочек (по умолчанию: 2.0)
- **Минимальная длина палочки** (по умолчанию: 10 px)
- **Максимальная длина палочки** (по умолчанию: 100 px)
