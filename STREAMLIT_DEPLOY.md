## 🌐 Шаги для публикации на Streamlit Cloud:

### Шаг 1: Перейдите на Streamlit Cloud
Откройте: https://share.streamlit.io/

### Шаг 2: Войдите через GitHub
- Нажмите "Sign in with GitHub"
- Выберите ваш аккаунт: halmuratdias@gmail.com

### Шаг 3: Создайте новое приложение
1. Нажмите "New app"
2. Выберите репозиторий: `Microbiologiaa`
3. Основной файл: `app.py`
4. Нажмите "Deploy"

### Шаг 4: Ожидание развертывания
- Развертывание занимает 2-5 минут
- Вы получите ссылку вида: `https://microbiologiaa.streamlit.app`
