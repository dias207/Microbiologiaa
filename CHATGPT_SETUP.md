# 🤖 Настройка ChatGPT классификации

## Что дает ChatGPT интеграция?

✅ **Точная идентификация бактерий** по микроскопическим изображениям  
✅ **Правильная таксономия** (семейство, род, вид)  
✅ **Обоснование результатов** с микробиологической точки зрения  
✅ **Высокая уверенность** в классификации  

## 📋 Инструкция по настройке

### 1. Получите API ключ OpenAI

1. Перейдите на https://platform.openai.com/api-keys
2. Войдите в аккаунт OpenAI
3. Нажмите "Create new secret key"
4. Скопируйте ключ (начинается с `sk-`)

### 2. Настройте проект

**Способ 1: Файл .env**
```bash
# Скопируйте шаблон
cp .env.example .env

# Откройте .env и замените ключ
OPENAI_API_KEY=sk-your-real-api-key-here
```

**Способ 2: Переменная окружения**
```bash
# Windows
set OPENAI_API_KEY=sk-your-real-api-key-here

# Linux/Mac
export OPENAI_API_KEY=sk-your-real-api-key-here
```

### 3. Перезапустите приложение

```bash
streamlit run app.py
```

## 🔄 Работа без API ключа

Если API ключ не настроен, система автоматически использует:
- **Эвристическую классификацию**
- **Анализ формы бактерий**
- **Определение по морфологическим признакам**

## 💰 Стоимость использования

- **GPT-4 Vision**: ~$0.01 за изображение
- **1 изображение** = ~5 тенге
- **100 изображений** = ~500 тенге

## 🔧 Проверка работы

После настройки:

1. **Запустите приложение**
2. **Загрузите изображение бактерий**
3. **Проверьте консоль** - должно появиться:
   ```
   ✅ ChatGPT классификатор инициализирован
   🤖 Используется ChatGPT для анализа
   ```

4. **Результат будет содержать:**
   - Точную таксономию
   - Уверенность в процентах  
   - Обоснование классификации

## 🆘️ Решение проблем

### Ошибка "API ключ не найден"
- Проверьте правильность файла `.env`
- Убедитесь, что ключ начинается с `sk-`
- Перезапустите приложение

### Медленная работа
- Проверьте подключение к интернету
- Попробуйте другое изображение
- Проверьте баланс API ключа

### Неправильная классификация
- Убедитесь, что изображение четкое
- Используйте изображения с хорошим освещением
- Попробуйте эвристический режим
